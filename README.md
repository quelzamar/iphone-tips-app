iPhone Tips App is a simple iOS application that provides useful tips and tricks
to help users get more out of their iPhone.

## Purpose
The goal of this project is to design a clean and lightweight iOS application while
exploring modern iOS development practices using Swift and SwiftUI.

## Features (Version 1)
- Display a list of iPhone tips
- Clean and minimal user interface
- Basic navigation

## Technologies
- Swift
- SwiftUI
- Xcode
